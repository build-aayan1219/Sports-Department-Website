# 🏏 Sports Department Website

A modern, responsive **Sports Department website** inspired by the Ministry of Youth Affairs & Sports (Government of India).
This project showcases sports initiatives, athlete highlights, news updates, and interactive UI components.

---

## 🚀 Features

* 🎨 **Modern UI Design** with Bootstrap 5
* 📱 **Fully Responsive Layout** (Mobile + Desktop)
* 🧭 Smooth navigation with sticky navbar
* 🖼️ **Interactive Gallery with Lightbox**
* 📰 Latest News & Updates section
* 💼 Vacancies / Careers section
* 📩 Functional Contact Form (frontend simulation)
* ⚡ Smooth scrolling and animations

---

## 🧩 Sections Included

* **Hero Section** – Intro banner with call-to-action
* **Stats Section** – Key achievements
* **Government Schemes** – Khelo India, FIT India, etc.
* **Gallery** – Athlete showcase with lightbox view
* **News & Updates** – Dynamic styled content
* **Vacancies** – Job listings
* **Contact Form** – User interaction
* **Footer** – Contact details and info

---
